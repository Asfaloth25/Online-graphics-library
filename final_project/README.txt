
Para ejecutar el programa, es necesario instalar ModernGL, Flask, RestrictedPython y Pillow.
Pueden instalarse mediante el comando:

    pip install moderngl RestrictedPython pillow flask

Una vez instalados, se puede ejecutar el programa desde la carpeta raíz con el siguiente comando:

    python .\final_project\main_server.py

Normalmente, la aplicación será lanzada en la dirección:

    http://127.0.0.1:5000
